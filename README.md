# Premier League Database System

A group project developed as part of a university database systems coursework. The application uses **PHP** and **MySQL** to manage and query Premier League 2022/23 data through a dynamic web interface.

The project demonstrates relational database design, SQL querying, and backend web development.

---

## Features

- Relational database design
- MySQL database implementation
- Dynamic PHP web interface
- Player and club management
- Club ownership and history records
- SQL queries using JOIN, GROUP BY, ORDER BY and LIMIT
- Dynamic data retrieval and display
